# GitHub 周趋势专场兴趣描述
# 每周日单独使用，用于从 GitHub / Hacker News / ProductHunt / 开发者社区中筛选值得回看的项目。

请重点关注以下内容：

1. GitHub Trending 中本周值得关注的新项目、快速增长项目、实用工具、开发者基础设施、AI Agent 框架、AI coding 工具、CLI、IDE 插件、浏览器自动化、数据处理、爬虫、MCP、工作流自动化、RAG、评测、模型部署、推理服务。
2. 有实际使用价值的开源项目：能解决明确问题、README 清晰、可本地运行、有示例、有活跃维护迹象、适合个人效率或工程实践。
3. AI 与开发者工具交叉项目：Agent 编排、多智能体、代码生成、代码审查、测试生成、自动化办公、知识库、搜索、网页理解、文档处理。
4. 重要开源生态变化：热门框架发布、大版本更新、许可证变化、维护者动态、平台政策、开源商业化、开发者社区趋势。
5. ProductHunt、Hacker News、V2EX、掘金、少数派中和开源项目、开发者工具、AI 产品相关的高质量讨论。

降权和过滤：
- 不要纯营销项目、套壳产品、README 空洞、没有明确使用场景的项目。
- 不要普通招聘、课程广告、低质量资源合集。
- 如果只是泛泛的 GitHub 热度但缺少实际用途，请降低分数。

输出偏好：
- 优先选择可以实际试用、适合收藏、可能进入个人工作流的项目。
- 对每个项目尽量说明它解决什么问题、为什么值得看、适合什么场景。
